Real Python könyvajánlója

• Kezdőknek is jók:

Mark Lutz: Learning Python
Christian Mayer: Python one-liners Nagyon jó, kis gyakorlati példák részletes magyarázattal.
Al Sweigart: Automate the boring stuff with Python (2nd edition). Kezdőknek – de sok témát, sok modult érint, középhaladóknak is jó.
Paul Barry: Head First Python. Kezdőknek – a szokásos Head First stílus: nem fed le minden anyagot, de részletes és szórakoztató.
Python Notes for Professionals A címével ellentétben kezdő anyag is van benne; nagyon sok téma, rengeteg kis példával.
Mark Pilgrim: Dive into Python 3 Kezdőtől haladóig; a nehezebb témák is nagyon jó, egyszerű példákkal.
David Beazley, Brian K. Jones: Python Cookbook. A gyakorlatban előforduló sok probléma megoldásai, kezdőknek is jó az alapozás után.
Bruce Eckel: Python 3 Patterns, Recipes and Idioms. Hasonló a Cookbook-hoz; kevesebb és tipikusan haladóbb témákat érint – de azokat nagyon jól elmagyarázza.

• Inkább haladóknak valók:

Mark Lutz: Programming Python
Brett Slatkin: Effective Python. A címével ellentétben elsősorban nem optimalizálásról, hanem jó programozási módszerekről szól.

Dan Bader: Python Tricks; A Buffet of Awesome Python Features. Tipikus feladatok, problémák, megoldások; a nyelv felépítése, filozófiája; jó és rossz programozási módszerek; a leggyakrabban gondot okozó fogalmak részletes, érthető magyarázata.
Alex Martelli, Anna Ravenscroft, Steve Holden,: Python in a Nutshell. A nyelv és a standard library tulajdonságai; sok olyan is, amit máshol nem hangsúlyoznak eléggé.

    Keresési tippek

    ● Érdemes egy témára külön rákeresni

    • a Google-ban
    • a Stackoverflow-n, itt szűkebb témát is érdemes lehet beírni, mint pl. "Python iterator problem"
    • a Youtube-on
    • a Speaker Deck-en, beírva persze a keresőjébe, hogy Python, vagy valami más, nagyobb témát, pl. azt, hogy "Python sets"
    • érdemes Google-ban rákeresni az egyes évek Python-konferenciáira, pl.: PyCon 2016 – aztán lehet mazsolázni az előadások között.

    ● Ha csak bizonyos nyelvű, például magyar találatokra vagyunk kíváncsiak a Google keresésnél, akkor elindítjuk a keresést, aztán a találati oldalon: 

    Beállítások ⇒ Speciális keresés ⇒ nyelv
    majd a kívánt nyelv beállítása után legörgetünk az oldal aljára a Speciális keresés gombig és katt.

    ● A Youtube videók alatt gyakran van egy TÖBB (MORE) bekezdés, amit ha kinyitunk, hasznos dolgokat találhatunk, pl. a szöveges változatot, programkódot, stb.

    ● Ha a Google-ban egy keresőszó után beírjuk, hogy vs, akkor sokszor hasznos párosításokat kínál fel. Pl. beírjuk, hogy "Python list vs " és ezek jönnek fel:

    • Python list vs tuple
    • Python list vs array
    • Python list vs tuple speed
    • ...

    ● Amikor általában tájékozódunk, érdemes a témához (a fő kereső kifejezéshez) hozzáfűzni ilyen kiegészítéseket:

    • best, worst practices
    • patterns, antipatterns
    • tutorials, [practical] examples, explaining
    • speed, performance, effective
    • tips, tricks, recipes, pitfalls
    • how to
    • problems, sucks
    • ...
    pl.: Python list comprehension tricks.

    ● A keresésbe érdemes lehet beleírni, melyik fő Python verzió édekel, pl.: "Python 3 list comprehension".

    ● Ha valakinek tetszett egy könyve/cikke/videója, érdemes a nevére rákeresni a fentebb felsorolt fórumokon.
