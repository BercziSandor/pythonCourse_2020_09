    Keresési tippek

    ● Érdemes egy témára külön rákeresni

    • a Google-ban
    • a Stackoverflow-n, itt szûkebb témát is érdemes lehet beírni, mint pl. "Python iterator problem"
    • a Youtube-on
    • a Speaker Deck-en, beírva persze a keresõjébe, hogy Python, vagy valami más, nagyobb témát, pl. azt, hogy "Python sets"
    • érdemes Google-ban rákeresni az egyes évek Python-konferenciáira, pl.: PyCon 2016 – aztán lehet mazsolázni az elõadások között.

    ● Ha csak bizonyos nyelvû, például magyar találatokra vagyunk kíváncsiak a Google keresésnél, akkor elindítjuk a keresést, aztán a találati oldalon: 

    Beállítások ⇒ Speciális keresés ⇒ nyelv
    majd a kívánt nyelv beállítása után legörgetünk az oldal aljára a Speciális keresés gombig és katt.

    ● A Youtube videók alatt gyakran van egy TÖBB (MORE) bekezdés, amit ha kinyitunk, hasznos dolgokat találhatunk, pl. a szöveges változatot, programkódot, stb.

    ● Ha a Google-ban egy keresõszó után beírjuk, hogy vs, akkor sokszor hasznos párosításokat kínál fel. Pl. beírjuk, hogy "Python list vs " és ezek jönnek fel:

    • Python list vs tuple
    • Python list vs array
    • Python list vs tuple speed
    • ...

    ● Amikor általában tájékozódunk, érdemes a témához (a fõ keresõ kifejezéshez) hozzáfûzni ilyen kiegészítéseket:

    • best, worst practices
    • patterns, antipatterns
    • tutorials, [practical] examples, explaining
    • speed, performance, effective
    • tips, tricks, recipes, pitfalls
    • how to
    • problems, sucks
    • ...
    pl.: Python list comprehension tricks.

    ● A keresésbe érdemes lehet beleírni, melyik fõ Python verzió édekel, pl.: "Python 3 list comprehension".

    ● Ha valakinek tetszett egy könyve/cikke/videója, érdemes a nevére rákeresni a fentebb felsorolt fórumokon.
